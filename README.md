# Indeed Integration Assessment

## Overview
Excel-based analysis to check whether Indeed adds unique, relevant data roles compared to existing job sources.

## Key Insights
- 77% of roles were unique
- VIC & NSW had the highest unique roles
- QLD & SA had the strongest best-match roles

## Recommendation
Run a longer 8–12 week pilot before full integration.
